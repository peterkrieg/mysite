MediaScore—Instagram Analytics

I created this web app recently over the course of a month. It is an Instagram analytics tool that grades a user's content. The site connects to the Instagram API and is built on the MEAN stack. Technologies include: Instagram/Facebook/Twitter/Google maps API, bootstrap, sass, gulp, Highcharts.

I had a lot of fun with this project, and went pretty deep into it. Some of the most challenging parts were dealing with the massive amount of data from large Instagram accounts (up to 100+ MB of JSON in some cases). I had to bundle this data into a manageable size and use only the bare minimum necessary for the app to function. Another challenge was dealing with the conflicting DOM manipulation of Angular vs. highcharts.

Checkmate Pharmacueticals

I worked on this from May-July 2015. This website only deals with front end web development, but I spent a lot of time ensuring the site to look good on all devices, browsers, and screen sizes.

I created the website using the sass compoments and JS of Bootstrap 3. The initial logo was created by a graphic artist, but I was responsible for editing this logo later and tweaking the fonts and colors. I was responsible for the entire color scheme and look of the site.

Kronolearn

Kronolearn is a spaced repetition learning software that makes quiz-taking smarter. Users can create courses, and review questions based on the principles of spaced repetition. This was a group project for Dev Mountain, and built on top of the MEAN stack. Technologies include: bootstrap, gulp, sass, and AWS.

This project was a good challenge for getting better at git workflow with teams. Our group had nearly 500 commits, 40 branches, and 100 pull requests, and had to navigate the workflow efficiently.

Sly's Barber Shop

In the summer of 2014, I was looking for practice making websites. I realized the barbershop I went to didn't have a website and I thought they could benefit from one. I proposed free haircuts in exchange for the website--what a great deal!

This site is based heavily on a bootstrap template, and is messy code. But I came up with the design, logo, and all photography. I managed SEO and eventually had the page getting 50+ unique visitors per week. Sly actually noticed an increase in business which was great.
